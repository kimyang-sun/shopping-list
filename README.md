## 기능 설명

### [상품 추가 기능]

- 입력창에 상품을 입력하고 상품을 추가하는 기능입니다.

```
- 입력창에 상품을 입력 후 옆에 + 아이콘 클릭
- 입력창에 상품을 입력 후 Enter 키
```

### [상품 삭제 기능]

- 추가된 상품을 다시 삭제하는 기능입니다.

```
- 추가된 상품에 쓰레기통 아이콘 클릭
```
